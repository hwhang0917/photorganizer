# Photorganizer

#### Guide

1. Install Dependencies

   ```shell
   npm install
   ```

2. Start project

   ```shell
   npm start
   ```

#### Description

This is an image organizing web based application. This is a group project for the course Software Engineering (CSCI450) instructed by professor Grace Lew.
